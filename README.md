# 鸿蒙日历应用

一款基于HarmonyOS平台开发的日历应用。

## 📱 应用简介

这是一款基于ArkTS原生开发的日历应用，提供月视图、周视图、日视图三种展示模式，让用户能够从不同维度管理时间。  

应用精确计算并展示二十四节气，完整呈现天干地支纪年系统，同步显示农历信息与传统节日。

## ✨ 核心功能

### 🗓️ 多视图日历展示
- **月视图**：完整月份概览，支持快速浏览，双历融合显示公历与农历日期
- **周视图**：网格化布局，时间轴与星期交叉显示，同时段事件动态适配高度
- **日视图**：详细时间轴展示，同时段事件横向排列，优化空间利用

### 🏮 传统文化集成
- **24节气显示**：基于精确算法计算传统二十四节气，特殊标识展示
- **干支纪年系统**：完整的天干地支年月日计算，以2025/12/21冬至为基准
- **农历信息**：准确的农历日期对照显示
- **传统节日**：重要传统节日特殊标注

### 📅 日程管理功能
- **事件创建**：支持标题、描述、时间设置，24小时制时间选择
- **提醒功能**：基于JavaScript定时器的可靠提醒系统
- **数据持久化**：使用HarmonyOS Preferences API实现本地数据存储
- **事件编辑**：完整的增删改查操作，响应式UI更新

## 🛠️ 技术架构

## 🎯 设计特色

### 界面设计亮点
在月视图界面中，设计了双历融合的日历组件，将公历日期与传统农历日期并行展示，为用户提供完整的时间维度体验。

界面对传统节日和二十四节气采用了差异化的视觉呈现，提升用户的使用体验。

日历下方特别设置了信息详情区域，当用户选中特定日期时，该区域会展现选中日期的详细农历信息，包括完整的干支纪年、纪月、纪日体系。

为了进一步增强界面的视觉吸引力和现代感，在空余区域融入了装饰元素，填补界面的留白空间，为整体设计增添了一些生动活泼的氛围。

### 技术创新点
- **精确节气计算**：使用预定义的2024-2026年准确日期数据，确保节气显示的准确性
- **干支纪年算法**：以2025/12/21冬至日为基准点实现完整的干支计算
- **响应式UI更新**：采用@Watch监听器和版本号机制确保界面及时刷新
- **智能事件布局**：日视图中同时段事件横向排列，周视图动态单元格高度
- **本地数据持久化**：使用HarmonyOS Preferences API实现数据持久化

## 📱 使用说明

### 基本操作
1. **切换视图**：点击底部的"月"、"周"、"日"按钮切换不同视图模式
2. **添加日程**：点击右上角的"+"按钮或点击空白时间槽创建新日程
3. **查看日程**：在各个视图中点击日程事件可查看详情
4. **编辑日程**：点击日程后在弹出的对话框中编辑信息
5. **删除日程**：在编辑对话框中点击"删除"按钮移除日程

### 日程设置选项
- **标题**：输入日程的标题（必填项）
- **描述**：添加日程的详细描述信息（可选）
- **时间**：设置开始和结束时间，支持24小时制
- **全天事件**：开启全天事件开关
- **提醒设置**：选择合适的提醒时间（准时、5分钟前、10分钟前等）
- **颜色标记**：选择日程的显示颜色进行分类

### 传统文化功能
- **农历查看**：在月视图中查看每日对应的农历日期
- **节气提醒**：二十四节气以红色文字特殊标识
- **干支信息**：选中日期后查看详细的干支纪年、纪月、纪日信息
- **传统节日**：重要传统节日自动标注显示
