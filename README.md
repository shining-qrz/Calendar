# 鸿蒙日历应用

一个功能完整的鸿蒙日历应用，支持多种视图模式和日程管理功能。

## 功能特性

### 📅 多视图模式
- **月视图**: 显示整月日历，可查看每日事件概览
- **周视图**: 显示一周的详细时间安排
- **日视图**: 显示单日的详细日程安排

### 📝 日程管理
- **添加日程**: 支持创建新的日程事件
- **编辑日程**: 修改已有日程的详细信息
- **删除日程**: 删除不需要的日程
- **查看日程**: 查看日程的详细信息

### ⏰ 提醒功能
- **多种提醒时间**: 支持准时、5分钟前、10分钟前、15分钟前、30分钟前、1小时前、1天前等多种提醒选项
- **系统通知**: 集成系统提醒服务，确保不错过重要日程

### 🎨 个性化设置
- **颜色标记**: 为不同类型的日程设置不同颜色
- **全天事件**: 支持全天事件的创建和管理
- **详细描述**: 为日程添加详细的描述信息

## 技术特点

### 🏗️ 架构设计
- **MVVM架构**: 采用Model-View-ViewModel架构模式
- **服务分离**: 数据服务和提醒服务分离，便于维护
- **组件化**: 各个视图组件独立，可复用性强

### 💾 数据存储
- **本地存储**: 使用鸿蒙Preferences API进行本地数据持久化
- **数据同步**: 实时同步数据变更到本地存储

### 🔔 系统集成
- **通知权限**: 集成通知权限，支持应用内提醒
- **定时提醒**: 使用JavaScript定时器实现提醒功能

## 开发环境

- **DevEco Studio**: 6.0.1 Release版本
- **API Level**: 适配最新鸿蒙API
- **测试设备**: Mate70 Pro虚拟机

## 使用说明

### 基本操作
1. **切换视图**: 点击顶部的"月"、"周"、"日"按钮切换不同视图
2. **添加日程**: 点击右上角的"+"按钮创建新日程
3. **查看日程**: 在各个视图中点击日程可查看详情
4. **编辑日程**: 点击日程后在弹出的对话框中编辑
5. **删除日程**: 在编辑对话框中点击"删除"按钮

### 日程设置
- **标题**: 输入日程的标题（必填）
- **描述**: 添加日程的详细描述（可选）
- **时间**: 设置开始和结束时间
- **全天**: 开启全天事件开关
- **提醒**: 选择合适的提醒时间
- **颜色**: 选择日程的显示颜色

## 权限说明

应用需要以下权限：
- `ohos.permission.PUBLISH_AGENT_REMINDER`: 用于设置日程提醒通知

## 安装运行

1. 使用DevEco Studio 6.0.1打开项目
2. 连接Mate70 Pro虚拟机或真机
3. 点击运行按钮进行编译和安装
4. 首次运行时需要授予提醒权限

## 注意事项

- 确保设备系统版本支持所使用的API
- 首次使用时需要手动授予提醒权限
- 建议在真机上测试提醒功能的完整体验
